# 📝 To-Do List — Flutter

Aplicativo mobile de lista de tarefas desenvolvido com Flutter, utilizando gerenciamento de estado com `StatefulWidget`.

---

## 📱 Funcionalidades

- Adicionar tarefas via campo de texto
- Marcar tarefas como concluídas (checkbox com texto riscado)
- Deletar tarefas individualmente (ícone de lixeira)
- Interface limpa e responsiva com Material Design

---

## 🛠️ Tecnologias

- [Flutter](https://flutter.dev/)
- [Dart](https://dart.dev/)
- Material Design (pacote `flutter/material.dart`)

---

## 🚀 Como executar

### Pré-requisitos

- Flutter SDK instalado → [Guia de instalação](https://docs.flutter.dev/get-started/install)
- Emulador Android/iOS ou dispositivo físico conectado

### Passos

```bash
# Clone o repositório
git clone https://github.com/LuanBasani/sa02_todolist.git

# Acesse a pasta do projeto
cd sa02_todolist

# Instale as dependências
flutter pub get

# Execute o app
flutter run
```
