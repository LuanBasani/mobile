# Introdução ao Desenvolvimento Mobile

### Tipo de Desenvolvimento

- Nativo
    - Android:
        - SDK : Android SDK
        - IDE : Android Studio
        - Linguagens : Kotlin e Java
        - Ambientes : Mac, Win, Linux

    - IOS:
        - SDK : Cocoa Touch
        - IDE : Xcode
        - Linguagens : Swift / Objectype-C
        - Ambientes: Mac

- Multiplataforma
    - React Native :
        - SDK : Node.JS
        - IDE : VSCode, outros
        - Linguagens : JavaScript / TypeScript
        - Ambientes : Mac, Win, Linux

    - Flutter
        - SDK : Flutter SDK
        - IDE : VSCode, Android Studio
        - Linguagens : Dart
        - Ambientes : Mac, Win, Linux
